## Közzététel GitHub Pages-en

1. Töltsd fel ezeket a fájlokat a repó gyökerébe (vagy a `/docs` mappába).
2. **Settings → Pages → Source**: válaszd a `main` branch `/ (root)` (vagy `/docs`) opciót.
3. Pár perc múlva a `https://<felhasznalo>.github.io/<repo>/` címen elérhető lesz a belépő oldal.

A dokumentumok lényegében önállóak (a képek beágyazva vagy külső tárhelyről töltődnek);
nincs szükség build lépésre.
